## dataset Preparation
Create a new folder in directory called ```dataset``` and put the files from [https://cseweb.ucsd.edu/~jmcauley/datasets.html#steam_data](https://cseweb.ucsd.edu/~jmcauley/datasets.html#steam_data). There will be 5 files with the names ```australian_user_reviews.json.gz```, ```australian_user_reviews.json.gz```, ```bundle_data.json.gz```, ```steam_games.json.gz```, and ```steam_reviews.json.gz```.
